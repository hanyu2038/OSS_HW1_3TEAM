# 오픈소스 소프트웨어 기초 3팀

## 1. 프로젝트 소개

본 프로젝트는 **경북대학교 기숙사 내 학습 공간의 효율적이고 공정한 운영**을 위한 규칙 제정 프로젝트입니다.  
기숙사생들이 실제로 겪는 불편함과 개선 요구사항을 반영하여, 모두가 쾌적한 환경에서 학습할 수 있도록 체계적인 규칙을 수립하였습니다.

### 1-1. 프로젝트 목표

- 기숙사 학습 공간의 **공정하고 효율적인 이용 체계** 확립
- 정숙, 협업, 노트북 전용 공간 등 **다양한 학습 형태 지원**
- 소음, 자리 점유, 위생 문제 등 **실질적 불편 사항 해결**
- **오픈소스 협업 방식**을 통한 투명한 규칙 제정

## 2. 프로젝트 개요 및 목적

### 2-1. 주제
생활 속에서 느끼는 불편함과 개선점(**기숙사 학습 공간 운영**)을 바탕으로 규칙을 작성합니다.

### 2-2. 규칙 제정의 목적
기숙사 내 학습 공간을 모든 기숙사생이 **공정하고 쾌적하게** 이용할 수 있도록 필요한 기준과 절차를 확립하고, **기숙사생의 참여와 투명한 개정 과정**을 보장하는 것을 목적으로 합니다.

## 3. 규칙 문서 구성

본 프로젝트에서 작성하는 규칙 문서는 다음의 목차를 따르며, 최종 규칙 문서는 별도의 파일(`OSS_HW1_3TEAM.md`)로 관리됩니다.

### 3-1. 규칙, 조례 제목 및 목적
- 학습권 보장, 학습 공간의 효율적 활용, 다양한 학습 형태 지원, 공동체 생활의 질 향상

### 3-2. 적용 대상 및 범위
- **적용 대상**: 경북대학교 기숙사생, 관리자(기숙사 행정실 등)
- **적용 범위**: 정숙 공간, 협업 공간, 노트북 전용석, 개방형 학습 라운지
